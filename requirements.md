# みかんプロジェクト ウェブサイト 要件定義書

## 1. プロジェクト概要

### 1.1 プロジェクト名
みかんプロジェクト コーポレートサイト

### 1.2 目的
映像制作会社「株式会社みかんプロジェクト」の公式ウェブサイトを構築し、会社情報、スタッフ情報、制作実績を公開する。

### 1.3 対象ユーザー
- テレビ番組制作の依頼を検討している企業・団体
- 番組の視聴者・ファン
- メディア関係者
- 求職者

## 2. 機能要件

### 2.1 ページ構成

#### 2.1.1 トップページ（表紙）
- **URL**: `/`
- **内容**:
  - 会社のコンセプトメッセージ
    - 「みかんプロジェクトは映像制作会社です」
    - 「世の中に埋もれた大切な忘れものを届けることが出来たら」
    - 「そんな思いで番組を作っています」
  - ナビゲーション（Works / Staff / Company）

#### 2.1.2 Worksページ（制作実績）
- **URL**: `/works`
- **内容**:
  - NHK番組の制作実績一覧
    - ETV特集
    - 4K特集
    - 日曜美術館
    - その他
  - 各作品の詳細情報（放送日、タイトル、概要、関連リンク）
  - 年別・カテゴリ別のフィルタリング機能

#### 2.1.3 Staffページ（スタッフ紹介）
- **URL**: `/staff`
- **内容**:
  - ディレクター・プロデューサー 伊槻雅裕のプロフィール
  - 経歴・略歴
  - これまでの主な仕事
    - ラジオ番組
    - NHKテレビ番組（年別に整理）
  - 受賞歴・書籍化情報

#### 2.1.4 Companyページ（会社情報・お問い合わせ）
- **URL**: `/company`
- **内容**:
  - 会社概要
    - 商号: 株式会社みかんプロジェクト
    - 設立: 2020年10月
    - 資本金: 500万円
    - 業務内容: テレビ番組等の映像作品企画・制作
    - 代表取締役: 伊槻雅裕
  - お問い合わせフォーム
    - お名前（必須）
    - メールアドレス（必須）
    - お問い合わせ内容（必須）
    - 送信ボタン

### 2.2 共通機能

#### 2.2.1 ナビゲーション
- 全ページ共通のヘッダーナビゲーション
- レスポンシブ対応（モバイルメニュー）
- アクティブページのハイライト表示

#### 2.2.2 フッター
- 会社情報
- コピーライト表示

#### 2.2.3 レスポンシブデザイン
- デスクトップ（1920px以上）
- タブレット（768px - 1919px）
- モバイル（767px以下）

## 3. 非機能要件

### 3.1 パフォーマンス
- ページ読み込み時間: 3秒以内
- 画像の最適化（WebP形式、遅延読み込み）
- コード分割によるバンドルサイズの最適化

### 3.2 セキュリティ
- HTTPS通信の強制
- CSRF対策
- XSS対策
- SQLインジェクション対策
- 入力値の検証・サニタイズ

### 3.3 アクセシビリティ
- WCAG 2.1 AAレベル準拠
- キーボードナビゲーション対応
- スクリーンリーダー対応
- 適切なコントラスト比の確保

### 3.4 SEO
- 適切なメタタグ設定
- 構造化データ（JSON-LD）の実装
- セマンティックHTMLの使用
- サイトマップの生成

### 3.5 ブラウザ対応
- Chrome（最新版）
- Firefox（最新版）
- Safari（最新版）
- Edge（最新版）

## 4. 技術要件

### 4.1 使用技術
- **バックエンド**: Laravel 12
- **フロントエンド**: Bladeテンプレート + JavaScript（必要に応じて）
- **データベース**: MySQL 8.0
- **コンテナ**: Docker / Docker Compose
- **パッケージ管理**: Composer
- **CSS**: Tailwind CSS または カスタムCSS（BEM命名規則）

### 4.2 開発環境
- Docker Composeによる環境構築
- ホットリロード対応
- ローカル開発サーバー

### 4.3 デプロイ環境
- 本番環境の詳細は後日決定
