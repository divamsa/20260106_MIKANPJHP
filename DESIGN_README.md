# みかんプロジェクト ウェブサイト 設計書まとめ

## 概要

4つのPDFファイル（表紙.pdf、Company.pdf、Staff.pdf、Works.pdf）を基に、株式会社みかんプロジェクトのコーポレートサイトを構築するための設計書です。

## 設計書一覧

### 1. 要件定義書 (`requirements.md`)
- プロジェクト概要
- 機能要件（4ページの詳細仕様）
- 非機能要件（パフォーマンス、セキュリティ、アクセシビリティ等）
- 技術要件（Laravel 12、Docker、Composer）
- データ要件
- 運用要件

### 2. システム設計書 (`system_design.md`)
- システムアーキテクチャ
- ディレクトリ構造
- 技術スタック詳細
- データフロー
- セキュリティ設計
- パフォーマンス最適化
- エラーハンドリング
- テスト戦略

### 3. データベース設計書 (`database_design.md`)
- データベース概要
- ER図
- テーブル定義（works, staff, contacts）
- マイグレーション定義
- シーダー定義
- データ整合性
- パフォーマンス考慮事項

### 4. API設計書 (`api_design.md`)
- API概要
- エンドポイント一覧
- エラーハンドリング
- セキュリティ
- テスト

## ページ構成

1. **トップページ（/）**
   - 会社のコンセプトメッセージ
   - ナビゲーション

2. **Worksページ（/works）**
   - NHK番組の制作実績一覧
   - カテゴリ別・年別フィルタリング

3. **Staffページ（/staff）**
   - ディレクター・プロデューサーのプロフィール
   - 経歴・実績

4. **Companyページ（/company）**
   - 会社概要
   - お問い合わせフォーム

## 技術スタック

- **バックエンド**: Laravel 12
- **フロントエンド**: Bladeテンプレート + Tailwind CSS（またはカスタムCSS）
- **データベース**: MySQL 8.0
- **キャッシュ**: Redis
- **Webサーバー**: Nginx
- **コンテナ**: Docker / Docker Compose
- **パッケージ管理**: Composer
